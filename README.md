# MilaTec - Portal do Cliente

Plataforma web para acompanhamento de orçamentos, projetos, entregas e instalações da MilaTec.

## Estrutura

- `backend/` — API REST em NestJS integrada ao Airtable
- `frontend/` — Interface web em Vue.js

## Como rodar

### Backend
```bash
cd backend
npm install
npm run start:dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
