# MilaTec

Front-end em Vue 3 + Vite seguindo as diretrizes de identidade da MilaTec.

## Scripts

- `npm install` — instala dependências.
- `npm run dev` — sobe o ambiente em http://localhost:5173.
- `npm run build` — gera artefatos de produção em `dist/`.
- `npm run preview` — pré-visualiza o build.

## Padrões adotados

- Vue 3 com Vue Router.
- Fonte Poppins global.
- Layout corporativo alinhado à paleta MilaTec (azul/verde/branco).
- Componentes reutilizáveis (cards, seções, shell).
- Dados mockados em `src/mocks/` prontos para troca por API futura.
